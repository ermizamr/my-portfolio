Space News Telegram Bot

A Python-based automated bot that fetches daily space news from Spaceflight News API and posts it to a Telegram channel. Works both locally and on cloud platforms like Render.com.

Python Telegram Bot API Spaceflight News API Automation Render.com Cron Jobs

Key Features

Works Everywhere

Designed to work both locally on your laptop and on cloud platforms like Render.com with cron job scheduling for daily automated runs.

Fetches Latest News

Connects to Spaceflight News API to fetch the latest 5 space news articles, filtering for today's publications in Ethiopia timezone.

Automated Telegram Posts

Automatically sends formatted news articles to a specified Telegram channel with titles and direct links.

Environment Configuration

Uses environment variables with .env file support for secure configuration of Telegram bot token and channel ID.

Error Handling

Comprehensive error handling for API failures, network issues, and Telegram communication problems with detailed logging.

Timezone Aware

Uses Ethiopia timezone (Africa/Addis_Ababa) for filtering today's news, with proper datetime handling across timezones.

def fetch_latest_space_news():
    """Fetches today's space news articles from the Spaceflight News API."""
    url = "https://api.spaceflightnewsapi.net/v4/articles/?limit=5"
    try:
        response = requests.get(url)
        response.raise_for_status()
        data = response.json()

        today = datetime.now(ETHIOPIA_TZ).date()

        today_articles = [
            article
            for article in data["results"]
            if datetime.fromisoformat(article["published_at"].replace("Z", "+00:00"))
            .astimezone(ETHIOPIA_TZ)

How It Works

1

Environment Setup

The bot loads configuration from environment variables or a .env file, including the Telegram bot token and channel ID. It uses pytz for timezone handling with Ethiopia timezone as default.

2

Fetch Space News

Makes an API request to Spaceflight News API to fetch the latest 5 articles. Filters results to only include articles published today in the Ethiopia timezone.

3

Process & Format

Processes the API response, extracts article titles and URLs, and formats them for Telegram with emojis and clean formatting for readability.

4

Send to Telegram

Uses the python-telegram-bot library to send each article as a separate message to the configured Telegram channel. Includes error handling for network issues.

5

Logging & Monitoring

Comprehensive logging throughout the process with timestamps and error levels. When deployed on Render.com, logs are available in the dashboard for monitoring.
